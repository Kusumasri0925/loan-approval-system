# Frontend - Loan Approval System

This is the frontend application built using **React.js**.

It provides a user interface for applying and managing loans.

---

## Technologies

* React.js
* Tailwind CSS
* Axios
* React Router

---

## Pages

### Authentication

* Login
* Register
* Forgot Password
* Reset Password

---

### Dashboard

Shows:

* User CIBIL score
* Applied loans
* Approved loans
* Rejected loans
* Available loan types

---

### Loan System

Users can:

* Apply for loan
* View loan history
* Check eligibility

---

## Run Frontend

```
npm install
npm start
```

App runs on:

```
http://localhost:3000
```
